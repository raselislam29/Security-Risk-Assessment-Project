# Enterprise Cybersecurity Risk Assessment Showcase

This project demonstrates a structured, organization-wide cybersecurity risk assessment across infrastructure, applications, and data assets. It includes a practical risk register, mitigation planning, and control gap mapping to both NIST CSF and SOC 2 requirements.

## Project Highlights

- Conducted end-to-end risk assessment across cloud/on-prem infrastructure, business applications, and sensitive data workflows.
- Created a risk register with likelihood, impact, risk scoring, owners, and mitigation strategies.
- Mapped security gaps to NIST CSF categories and SOC 2 Trust Services Criteria.
- Produced executive-ready outputs that support remediation prioritization and audit readiness.

## Assessment Scope

- Infrastructure: identity systems, endpoint security, network segmentation, vulnerability management, backup/DR.
- Applications: SDLC controls, authentication, secrets management, logging/monitoring, API security.
- Data: classification, retention, encryption, access controls, DLP and backup integrity.

## Methodology

1. Define business context and critical assets.
2. Identify threats, vulnerabilities, and existing controls.
3. Score risks using a 1-5 likelihood and 1-5 impact model.
4. Prioritize remediation based on risk score and business criticality.
5. Map gaps to NIST CSF and SOC 2 for governance and compliance alignment.

### Risk Scoring Model

- Likelihood (1-5): Rare to Almost Certain
- Impact (1-5): Negligible to Severe
- Risk Score: `Likelihood x Impact`
- Priority Bands:
	- 15-25: High
	- 8-14: Medium
	- 1-7: Low
